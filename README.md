# ASL Sign Language Prediction

## Overview
ASL Sign Language Prediction is a deep learning-based project designed to recognize American Sign Language (ASL) hand gestures using a Convolutional Neural Network (CNN). The system captures hand gestures through a webcam and predicts the corresponding sign in real time.

This project demonstrates how computer vision and deep learning can be used to recognize sign language gestures.

---

## Features
- Real-time hand gesture detection using webcam
- Deep Learning model based on Convolutional Neural Network (CNN)
- Accurate classification of ASL hand gestures
- Displays the predicted sign label
